=== Plugin Name ===
Tags: NextGEN, gallery, galeri, resim, image, önizleme, thumbnail, excerpt, özet

NextGEN Galeri uygulaması kullanılan yazıların özetlerine otomatik olarak önizleme resmi ekler.

== Description ==

Wordpress yazılarına <a href="http://wordpress.org/extend/plugins/nextgen-gallery/">NextGEN Gallery</a> uygulaması ile eklenen resimlerden ilkini; otomatik olarak, özet alanına girilen metnin başında gösterir. Bunun için temanızda özet alanlar the_excerpt() etiketi ile gösteriliyor olması gerekmektedir.

== Installation ==

1. nextgen-icin-onizleme klasörünü wp-content/plugins klasörüne yükleyiniz.
2. Eklentiler sayfasından eklentiyi etkinleştiriniz.
